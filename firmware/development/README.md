# <i data-lucide="play-circle"></i> DEVELOPMENT FIRMWARE (Testing)

This directory is the **Active Sandbox Tier** for Wee2-D2.

## <i data-lucide="info"></i> ACTIVE TESTING ONLY

All new features, configuration logic experiments, and iterative testing **SHALL** be performed here first.

### Use this Tier for:
*   Iterative feature research (e.g., the New Speed Multiplier).
*   Bench verification on spare hardware.
*   "Fail-safe" development that doesn't impact droid stability.

### Graduation:
Once verified as stable by the user, code is "Graduated" to the [**`../production/`**](../production/) directory for final deployment.
